How to Submit

Existing Links

If you have existing links, you can submit them directly on this page.

Writing for VJs Mag

If you wish to write articles for VJs Mag and earn revenue, please follow these steps:

  1. Read the Documentation: Visit our documentation page to understand our submission guidelines and requirements.
  2. Create a Private Repository: Create a private repository on GitHub.
  3. Add Access Permissions: Add editorial Guillaume Lauzier to your repository for review. You can find him at Guillaume Lauzier on GitHub.

We look forward to your contributions!
